Graphical Results

``` {r include = TRUE, echo = FALSE}
ggplot(data = ___, aes(x = ___, y = ___) ) +
geom_boxplot (notch=TRUE,outlier.colour = "red",outlier.size = 2, fill = "darkgray") +
stat_boxplot (geom ='errorbar') +
labs (x = "___", y = "___", title = "___") +
theme (plot.title = element_text(hjust = 0.5, size = 16) )
```

  • This makes a boxplot
  • If its not working, look in packages and click "ggplot2"

Numerical Results

``` {r include = TRUE, echo = FALSE}
favstats(___~___,data=___)
```

  • This prints Q1-Q4 and the Median

Inferential Results (Single)

``` {r include = TRUE, echo = FALSE}
t.test(___~___,data=___)
```

  • This is a T-Test
  • It will give you a P-Value
  • Generally < 0.05 is significant

Inferential Results (Multiple)

``` {r include = TRUE, echo = FALSE}
model1 <- aov(___~___,data=___)
summary.aov(model1)
plot(TukeyHSD(model1,conf.level = 0.95))
```

  • This will run a Anova and a TukeyHSD
  • Same Rules for P-Values
  • Anova tells you IF there is a difference, TukeyHSD shows you where it is
