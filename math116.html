Numbers explained by Words ( Numerical ~ Cat )

BoxPlot

```{r include=TRUE,echo=FALSE}
ggplot(data = ___, aes(x = ___, y = ___) ) +
geom_boxplot ( notch=TRUE , outlier.colour = "red" , outlier.size = 2 , fill = "darkgray" ) +
stat_boxplot ( geom = 'errorbar' ) +
labs ( x = "___" , y = "___" , title = "___" ) +
theme ( plot.title = element_text (hjust = 0.5, size = 16) )
```

  • Put this in "Graphical Results"

Fav Stats

```{r include=TRUE,echo=FALSE}
favstats( ___ ~ ___ , data= ___ )
```

  • Put this in "Numerical Results"

T Test

```{r include=TRUE,echo=FALSE}
t.test( ___ ~ ___ , data = ___ )
```

  • Put this in "Inferential"
  • This is used for compating TWO catagories ONLY.

One Way Anova

```{r include=TRUE,echo=FALSE}
model1 <- aov( ___ ~ ___ , data = ___ )
summary.aov(model1)
```

  • Put this in "Inferential"
  • This is used for compating multiple catagories.
  • This does NOT tell you where the difference is, only that there is a difference somewhere

Tukey HSD

```{r include=TRUE,echo=FALSE}
par(mar = c(3, 8.7, 2, 2))
plot(TukeyHSD(model1),las=2)
```

  • Put this in "Inferential" under the One Way Anova
  • This is used for compating multiple catagories.
  • This tells you where the difference is.
