Numbers explained by Words
( Numerical ~ Catagorical )

Red Words you NEED to fill, Green Words are for Aesthetics

BoxPlot

```{r include=TRUE,echo=FALSE}
ggplot(data = YOUR DATA, aes(x = CAT DATA, y = NUM DATA) ) +
geom_boxplot ( notch=TRUE , outlier.colour = "red" , outlier.size = 2 , fill = "darkgray" ) +
stat_boxplot ( geom = 'errorbar' ) +
labs ( x = "X LABEL" , y = "Y LABEL" , title = "TITLE" ) +
theme ( plot.title = element_text (hjust = 0.5, size = 16) )
```

  • Put this in "Graphical Results"

Fav Stats

```{r include=TRUE,echo=FALSE}
favstats( NUM DATA ~ CAT DATA , data= YOUR DATA )
```

  • Put this in "Numerical Results"

T Test

```{r include=TRUE,echo=FALSE}
t.test( NUM DATA ~ CAT DATA , data = YOUR DATA )
```

  • Put this in "Inferential"
  • This is used for compating TWO catagories ONLY.

One Way Anova

```{r include=TRUE,echo=FALSE}
model1 <- aov( NUM DATA ~ CAT DATA , data = YOUR DATA )
summary.aov(model1)
```
  • Put this in "Inferential"
  • This is used for comparing multiple catagories.
  • This does NOT tell you where the difference is, only that there is a difference somewhere

Tukey HSD

```{r include=TRUE,echo=FALSE}
par(mar = c(3, 8.7, 2, 2))
plot(TukeyHSD(model1),las=2)
```

  • Put this in "Inferential" under the One Way Anova
  • This is used for comparing multiple catagories.
  • This tells you where the difference is.
  • If you need to change the width to fit everything, see the Green Words
