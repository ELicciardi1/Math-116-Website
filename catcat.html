Numerical Results

``` {r include = TRUE, echo = FALSE}
table1 <- xtabs(~___ + ___, data=___)
colPerc(table1)
```

  • This prints the data as a percentage
  • If it doesnt look right try switching "col" to "row"

Inferential Results (Single)

``` {r include = TRUE, echo = FALSE}
chisq.test(table1)
chisqtestGC(table1)
fisher.test(table1)
```

  • The Chi Squared is like the Anova, it onyl tells you IF there is a difference
  • The Fisher Test will tell you where it is, look for the low P-Value or high Chi-Statistic

Inferential Results (Multiple)

``` {r include = TRUE, echo = FALSE}
chisq.test(table1)
chisqtestGC(table1)
FisherTable(table1,confL = 0.95,printAll = TRUE)
```

  • The fisher table loops the fisher test for all combinations
  • Look for a low P-Value or High Chi-Statistic
