Graphical Results

``` {r include = TRUE, echo = FALSE}
ggplot(___,aes(y=___,x=___)) +
geom_point(aes(col=Sex)) +
geom_smooth(method=lm) +
labs(title = "___",x = "___",y = "___") +
theme(plot.title = element_text(hjust = 0.5, size = 20))
```

  • This makes a scatterplot
  • The regression line is also inferential data.

Numerical Results

``` {r include = TRUE, echo = FALSE}
cor(___~___,data=___)
```

  • This gives you a number showing if the items are correlated
  • If there is NO difference it will be 0, it will always be between -1 and 1

Inferential Results (Model)

``` {r include = TRUE, echo = FALSE}
model.1 <- lm(___~___,data=___)
summary.lm(model.1)
```

  • This will print a summary with a P-Value

Inferential Results (Anova)

``` {r include = TRUE, echo = FALSE}
anova(model.1)
```

  • If there are multiple categories, repeat the above steps to make model.2+
  • The the syntax will be "anova(model.1 , model.1 ... )"
